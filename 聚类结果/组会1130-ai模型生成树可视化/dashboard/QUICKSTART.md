# 快速启动指南

## 第一步：安装依赖

```bash
cd dashboard
npm install
```

如果还没有安装Node.js，请先安装：
- 访问 https://nodejs.org/
- 下载并安装LTS版本

## 第二步：启动开发服务器

```bash
npm run dev
```

服务器会自动启动，并在浏览器中打开 http://localhost:3000

## 第三步：查看仪表板

仪表板包含4个交互式图表：

1. **一级主题演化** - 堆叠面积图（左上，大图）
2. **模型影响力** - 柱状图（右上，小图）
3. **创新性质演化** - 折线图（左下，小图）
4. **桑基流向图** - 基础模型到创新的映射（右下，大图）

## 功能特性

- ✨ 深色主题，护眼设计
- 🎨 渐变色彩，视觉美观
- 📱 响应式布局，支持不同屏幕
- 🖱️ 交互式提示，鼠标悬停查看详情
- 🔍 时间轴缩放，聚焦特定时间段
- 📊 SVG渲染，图表清晰可缩放

## 常见问题

### Q: npm install 失败？
A: 尝试使用国内镜像：
```bash
npm install --registry=https://registry.npmmirror.com
```

### Q: 页面显示"加载错误"？
A: 检查 `public/data/dashboard_data.json` 文件是否存在

### Q: 图表不显示？
A: 检查浏览器控制台是否有错误，确保所有依赖已正确安装

## 构建生产版本

```bash
npm run build
```

构建完成后，文件在 `dist/` 目录，可以部署到任何静态服务器。
